# Prison Break — Exploratory Data Analysis

Exploratory data analysis (EDA) of a dataset of helicopter prison escapes, using Python and pandas.

## Questions explored
- How have helicopter prison escapes changed over time?
- Which countries see them most?
- What patterns emerge (success rate, methods, recurring individuals)?

## Stack
`Python` · `pandas` · `matplotlib` · `Jupyter Notebook`

## What it demonstrates
- Data loading, cleaning and wrangling with pandas
- Aggregation & grouping to answer concrete questions
- Visualization of findings

## Run it
​```bash
pip install -r requirements.txt
jupyter notebook
​```
